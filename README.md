# MTProtoMax Metrics Viewer

Beautiful terminal metrics dashboard for MTProtoMax Telegram proxy server.

![Version](https://img.shields.io/badge/version-1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- Real-time metrics visualization
- User statistics (connections, traffic)
- Upstream connection statistics
- ME (Multiplexed Endpoint) monitoring
- Live auto-refresh mode
- Beautiful terminal UI with colors
